final-practiceUnit0

This project is a simplified text-based roleplaying game created for LaunchCodes JavaScript Unit 0 final practice project. 
The program allows the user to choose a character name, select a starting weapon, and then follows a short simple story scenario using the console output.

The code is organized into two sections:

Section One demonstrates skills from Values, Data Types, and Operations, Stringing Characters Together, and Control Structures and Logic. 

Section Two demonstrates skills from Building Arrays, Using Arrays, and Creating and Using Objects via a player inventory and character object example.

This project is not meant to be a full game, but is instead a collection of small code examples that illustrate important JavaScript coding concepts/skills 
from Unit 0 of LaunchCodes Foundation course.
